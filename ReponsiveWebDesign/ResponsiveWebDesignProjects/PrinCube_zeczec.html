▲ 超過三萬名消費者對於 PrinCube 的熱烈期待。

我們相信在未來的世界裡，個人化與獨特性是每個人所的追求的，創意與設計更無處不在生活之中。為了讓更多人能輕鬆獲得高品質的攜帶式列印體驗，讓我們向大家詳細介紹地表上最小、最易用、使用相容性最佳的個人化印刷神器 - PrinCube！

PrinCube 非常輕巧，可一手輕鬆掌握。讓你不論在禮物外包裝或個人物品上，都可以輕鬆快速的進行彩色印刷。透過 Wi-Fi 與手機連接，就可輕鬆上傳任何文字、圖像或設計。按下按鈕輕輕一滑，即可印出你獨一無二的專屬印記。

▲ PrinCube在手，印出你專屬的創意生活。


卡片、信封、瓦愣紙、皮革、金屬、塑料、布料甚至木(竹)板。PrinCube 可為你實現生活中絕大部分的想像及創意。

▲ 搭配合適墨水，您可以使用 PrinCube 在生活中大部分的產品及材料表面，輕鬆列印。


PrinCube 噴墨式的設計，可以讓你為生活配件或用品，增添更多個性化的設計與想像。ex. 鴨舌帽、瓦愣紙、皮包、紙杯...等。

▲ 在合適的狀況下，PrinCube 可在粗糙，彎曲和不平坦的產品表面上列印。


▲ 你也可以用 PrinCube 在皮膚上印刷臨時的紋身。在安全的使用方式下，任何有創意的想法，PrinCube 都可以為你實現。


革命性的設計，讓印刷從此不再枯燥乏味，侷限在辦公室中。PrinCube 是無線的，可以透過 Wi-Fi 操作。重量輕巧，容易攜帶。符合人體工學的設計，讓你一手就能輕鬆使用。重量僅有 160g (包含墨水匣)，是目前全球最輕巧的印刷機。

▲ PrinCube 的尺寸只有 72 x 51 x 68 (mm)。輕巧方便，可以放進口袋裡，帶去任何地方旅行，讓你的創作沒有極限。


實用及方便是 PrinCube 的核心設計理念。不論 iPhone 或 Android 都可以連接，輕鬆上手。當 PrinCube 透過掃描連上雲端 操作介面 後，這款手掌般大小的印刷機就準備好了，只需簡單4步驟，即可輕鬆完成你的個人化創意。

▲ PrinCube 可以透過 Wi-Fi 與您的手機或筆電連接，簡單地設定你想列印的圖像或文字。然後輕鬆按下 PrinCube 上的按鈕並輕輕滑動，就可以印出你的專屬設計。


市面上攜帶式印刷機，大部分僅限於黑白色列印的功能。PrinCube 是世界上最便利的的口袋式彩色印刷機。它採用最新的三色熱噴墨技術，提供生動的彩色印刷。從今天起，你豐富的創意與設計能量，可以用更完美的方式，呈現在日常生活的各個面向。

▲ PrinCube 色彩逼真還原度高，同時支援彩色列印。



PrinCube 支援多行列印功能。在一般模式下，最大列印高度為 14.29mm，一次可列印約 1.3公尺 長。而在多行列印模式下，最多可以列印約 3公尺 長。因此，若你有列印較大尺寸的圖案需求，您可以藉由輕巧的 PrinCube，開啟多行模式功能，配合輔助工具，將你的創意完美實現。

▲ PrinCube 依照你的創意需求，支援多行列印。


PrinCube 使用最新的電池規格。一次完整充電可以提供 6小時 的連續列印和 1年 的待機時間，是市面上其他移動式印刷機的 20倍，高續航的電力特色，讓 PrinCube 可以隨時待命，完美適應各種商務旅行及特殊的印刷需求。

▲ PrinCube 一次充電可以連續使用6小時和擁有1年的待機時間，隨時為你的創意待命。


PrinCube 的標準型墨水匣適合在紙製品上印刷，如同傳統的印表機一樣，墨水在紙張上的附著性極佳，只要沒有碰到水或被汗水浸濕，你的創作都可以完美保存。而如果你有在金屬、皮革等表面列印的需求，我們也推出了持久型墨水匣，它具有更好的耐久性以及基本的防水性。此外，一盒墨水匣可以列印 415張 的 A4紙！讓你印到手軟，創作不間斷！

★ 需要加購墨水匣？請直接選擇標準型或持久型兩種墨水匣的方案做贊助，屆時即會與你的 PrinCube 一起出貨。

▲ 當需要更換墨水匣時，只需打開 PrinCube，放入新的墨水匣，幾秒鐘內即可輕鬆完成，繼續列印。


PrinCube 的設計特色除了輕巧好攜帶、符合人體工程學之外。更將極簡理念發揮到極致，所有的列印操作都通過一個按鈕進行控制即可。並採用最新的 USB Type-C 接頭充電，符合現代行動生活方式的趨勢及需求，絕對是你前所未有的絕佳體驗。

▲ PrinCube 採用最新的 USB Type-C 接頭充電，一鍵按鈕即可輕鬆使用。


此外，我們設計了一個有趣的附加功能，讓 PrinCube 更加貼心獨特。它的 私密列印功能 允許兩個使用者之間 共享加密訊息。發送的信息只有在列印出來之後才會顯示，是情侶或好友之間傳情或分享秘密訊息的好方法。

▲ PrinCube 獨家的私密列印功能，讓你在生活中擁有更多驚喜。


PrinCube 身為領導品牌，了解智慧型產品，技術變化很快。因此 PrinCube 支援 API 二次開發，以滿足開發商現在和未來的獨特需求。在未來的升級中，我們將持續提供給所有用戶更多創新與方便的設計。

▲ PrinCube 支援 API 二次開發，為未來的升級做準備。
